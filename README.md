# Sorting Algorithms In Java

## Sorting Algorithms in this repository

* Insertion Sort
* Bubble Sort
* Selection Sort
* Shell Sort
* Merge Sort
* Quick Sort
* Heap Sort

#### All the methods in classes are static. You can just call them as shown below

```
Integer[] array = {28,15,48,32,190,1,-9};
BubbleSort.sort(array);
```
